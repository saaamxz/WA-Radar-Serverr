WA-Radar-Server 
سيرفر خلفي لمراقبة رسائل الواتساب المحذوفة وإعادة توجيهها إلى تيليجرام.
 نظرة عامة
هذا المشروع مصمم ليعمل كجسر بين واتساب وتيليجرام. يمكنك تشغيله في دقائق عبر Replit.
 التشغيل السريع
1.	استيراد المشروع: اضغط على زر Fork لنسخ المشروع إلى حسابك في Replit.
2.	إعداد المتغيرات: * من قائمة Secrets (أيقونة القفل)، أضف القيم التالية:
* TG_TOKEN: التوكين الخاص ببوتك.
* TG_CHAT_ID: الـ ID الخاص بك.
3.	التشغيل: اضغط على زر Run.
4.	الربط: سيظهر لك QR Code في الـ Console، امسحه من واتساب.
   واجهة المستخدم
يمكنك العثور على دليل إضافي وتفاصيل حول المشروع في ملف index.html الموجود داخل المجلد الرئيسي.
🛠 التقنيات المستخدمة
• Node.js: البيئة الأساسية للتشغيل.
• WhatsApp Web API: لمراقبة الرسائل.
• Telegram Bot API: لإرسال الإشعارات.
